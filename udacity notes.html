Inline & Block - most html elements are defined as either block or inline.
Inline is anything "in the line" without any breaks
Block elements basically put a invisible box around as a container for the content inside of that block.

The "br" tag inserts a single line break. It is an empty tag which means that it has no end tag. (I had to actually research how to insert the <br> tag into my note text without breaking the sentence as the break was intended to so.
